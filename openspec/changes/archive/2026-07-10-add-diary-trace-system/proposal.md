# Proposal: add-diary-trace-system

## Why

内测反馈暴露两个断层：价值兑现太慢（呼吸引导每次冷启动阻塞主页），以及体验链条在"说完了"之后就断了——用户聊出的内容沉入不可见的 storage，直到整本图鉴完成才一次性兑付。产品定位已重新确认：**聊聊是代笔日记，回顾是纪念册的跋**——AI 的职责是帮用户丰富记录，让未来翻看时唤起当时的记忆与情感。当前实现没有任何"翻看"的表面（锁定条目点击直接 `return`），日常任务层的对话不留痕（早期用户的日记页正在永久丢失，与 add-metrics-system 的紧迫性同构），照片以未压缩 base64 全量落库（storage 超限风险）。

## What Changes

- **呼吸引导改造**：仅首次启动强制；之后主页直接渲染，呼吸退为常驻"静一下"主动入口
- **完成一拍**：点"做完啦"后先有确认一拍（标本落册动效 + 一句指向体验而非评价的文案），聊聊邀请随后浮现，邀请文案含照片敞口
- **对话开场问细节**：开场白从"有什么感受吗"改为问具体细节；AI 采集记忆扳机（感官/地点/意外感），邀请深入一次、跟随用户能量
- **摘要=日记页**：摘要 prompt 保留用户原话措辞；新规则"无实质内容不成页"
- **重逢弹层（核心）**：图鉴已锁定条目点击展示日记页（日期 + "那天你说：…" + 照片缩略图）；同一弹层复用于"今日已完成/昨日完成"条目
- **BREAKING：日常任务层聊聊留痕**——翻转"推送层语义：退出不生成摘要"的既有决定，日常任务/即时任务的聊聊也生成并持久化摘要（验收单 4.5 语义翻转）
- **照片痕迹化**：归档时压缩缩略图存入痕迹，归档后裁掉对话中的原图 base64；明确排除上传服务器；照片只等被翻到、绝不主动推送
- **归档可见化**："说完了"收尾配"对话凝成一页落进册子"动效
- **三件幸福小事**：常驻轻入口（无领取/完成态、无未完成暗示），点击直接进专用对话（聊即是做），文案内置"想到一件也算数"；内容文案过 Gate 0
- **回顾重定位为跋**：回顾页展示该图鉴收到的照片小图集；册子正文是各条目日记页

设计红线（全程适用）：不制造焦虑、数值不可见、见证者可寻而不索取、记忆不追人、做歪了也算数。

**明确不在本变更内**（记录在案的待决事项）：手记页（完整时间轴翻阅面）上线后凭数据设计，约束已定——册页式陈列、无空格子、日历只做跳转导航不做成就墙、防流水账；照片 storage 预算与淘汰策略细化。

## Capabilities

### New Capabilities

- `diary-trace`: 日记痕迹数据层——摘要即日记页（保留原话、无实质不成页）、日常任务层聊聊生成摘要、照片缩略图入痕与原图裁剪、"清掉"只清视觉不删数据
- `trace-reencounter`: 重逢弹层——图鉴锁定条目与今日/昨日完成条目点开可见日记页（日期 + 原话 + 照片），取代锁定条目的死点击
- `completion-beat`: 完成一拍与归档可见化——"做完啦"后的确认节拍、聊聊邀请措辞（含照片敞口）、"说完了"落册动效
- `diary-conversation`: 对话的日记采集姿态——开场问细节不问感受、记忆扳机追问、一次邀请不索取、结束权永远在用户
- `breathing-entry`: 呼吸引导仅首次启动强制，之后为常驻"静一下"主动入口
- `three-good-things`: 三件幸福小事常驻轻入口——聊即是做、专用开场、无完成态无配额压力

### Modified Capabilities

- `instant-task`: ①"退出不生成摘要"的推送层语义翻转为生成日记痕迹；②"呼吸引导完成后"前置条件改为仅首次启动
- `collection-review`: ①回顾页新增该图鉴照片小图集；②回顾定位为"跋"（素材收集与快照机制不变）
- `daily-tasks`: 完成流程插入完成一拍；完成条目（今日/昨日）点开可见日记痕迹

## Impact

- **组件**：`src/pages/index/index.vue`（呼吸门控、静一下入口、幸福小事入口、完成条目弹层）、`src/components/CollectionDetail.vue`（锁定条目重逢弹层）、`src/components/ChatView.vue`（照片压缩、开场、动效）、`src/components/ReviewView.vue`（照片小图集）、新增重逢弹层组件
- **状态/存储**：`src/state/conversation.js`（归档时缩略图生成与原图裁剪）、`src/state/storage.js`（日常层摘要持久化；"清掉"语义分离）
- **API prompt**：`src/api/deepseek.js`（buildSummaryPrompt 原话保留 + 无实质不成页）、`src/api/qwen.js`（开场白与记忆扳机姿态、幸福小事专用开场）
- **文档/验收**：`manual_acceptance_checklist_v2.md` 4.5 语义翻转及新增验收项；内容新文案过 Gate 0 审核流程
- **不影响**：analytics-events（上报载荷不变，摘要/照片仍禁止上报）、Supabase 服务端、回顾快照机制
